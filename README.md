# URL-MANAGEMENT-SYSTEM
A secure and user-friendly Spring Boot REST API application that allows users to register, authenticate, and manage their personal URLs. The system uses JWT-based authentication and ensures each user can only access their own data.

# 🚀 Features
🔐 User Registration & Login with JWT Authentication
🔑 Password Encryption using BCrypt
🌐 Add, View, Update, Delete URLs
👤 User-specific URL isolation (secure multi-user system)
🧾 Profile update and account deletion
🛡️ Secure REST APIs using Spring Security
⚡ Stateless authentication (JWT-based)

# 🛠️ Tech Stack
Backend: Java, Spring Boot
Security: Spring Security, JWT
Database: PostgreSQL / MySQL / H2
Build Tool: Maven
API Testing: Postman
